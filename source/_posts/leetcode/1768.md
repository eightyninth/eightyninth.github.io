# 解法
```python
def mergeAlternately(word1, word2):
    ret = ""
    i, a_len, b_len = 0, len(word1), len(word2)
    while i < a_len and i < b_len:
        ret += word1[i] + word2[i]
        i += 1
    if i < a_len: ret += word1[i:]
    if i < b_len: ret += word2[i:]
    return ret
```
