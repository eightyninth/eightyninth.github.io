# 解法：hash表

```python
def countMatches(items, ruleKey, ruleValue):
    ret = 0
    keys = {"type": 0, "color": 1, "name": 2}
    for item in items:
        if item[keys[ruleKey]] == ruleValue:
            ret += 1
    return ret
```
