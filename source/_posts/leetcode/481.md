# 解法： 暴力

```python
def magicalString(n):
    s = "122"
    if n <= 3:
        return 1
    ret, i, add = 1, 2, "1"
    while i < n:
        if s[i] == "1":
            ret += 1
        s += add * int(s[i])
        add = "1" if add == "2" else "2"
        i += 1
    return ret
```
